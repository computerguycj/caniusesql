# caniusesql.com

A [caniuse.com](https://caniuse.com)-style reference for SQL feature compatibility across databases.

## What is it?

Quickly check whether a SQL feature is supported in PostgreSQL, MySQL, SQLite, SQL Server, MariaDB, and more — without digging through individual database docs.

## Status

Work in progress. Functional but actively being developed.
